Available methods inside jobs

wqLog([parameters]) Helps jobs debugging by sending a message with parameters to the DOM. See aci-js-panel/demo for an example.
wqLog([result], {options})

Registers the current job result. Options:

reqNewJob boolean If true, once the result is registered, the server will assign a new job.
